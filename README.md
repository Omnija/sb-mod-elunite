# sb-mod-elunite
Elunite Race Mod for Starbound

Author: Omnija
Assistant: Sotaeko

Description: 
Adds Elunite as a playable race.

Installation: 
Move the elunite.pak or folder into your Starbound/mods/ folder. 

Mod Dependency:
You need an Extended Race Selection mod for this to work.
